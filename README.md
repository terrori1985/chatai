# AI Chat Assistant

Графическое приложение для общения с локальными LLM моделями (Ollama) и Google Gemini с поддержкой голосового ввода, синтеза речи и голосовых команд управления ПК.

## 📋 Возможности

### 🤖 Работа с AI моделями
- Поддержка локальных моделей через **Ollama**
- Поддержка **Google Gemini API**
- Потоковая генерация ответов (streaming)
- Возможность прикрепления файлов к запросам
- Настраиваемое форматирование ответов (очистка от Markdown)

### 🎙 Голосовые функции
- **Голосовой ввод** с настройкой чувствительности микрофона
- **Синтез речи** (озвучивание ответов AI)
- **Разговорный режим** (автоматический цикл: слушать → ответить → озвучить)
- **Голосовые команды управления ПК** (открытие приложений, управление медиа, громкостью и т.д.)

### 🎨 Интерфейс
- Современный дизайн с поддержкой тем (светлая, темная, пользовательская)
- Настраиваемая прозрачность окна
- Разделение истории на "Ваши сообщения" и "Ответы модели"
- Компактный виджет поверх всех окон
- Системный трей для работы в фоне

### 📄 Работа с файлами
- Прикрепление файлов к запросам
- Просмотр содержимого файлов
- Определение кодировки файлов

### ⚙️ Настройки
- Подключение к серверу Ollama
- Настройки звуковых уведомлений
- Настройки Gemini API
- Форматирование ответов AI
- Чувствительность микрофона
- Управление голосовыми командами

## 🚀 Установка

### Требования

- **Python 3.8+**
- **Ollama** (для локальных моделей)
- **pip** (менеджер пакетов)
